# 🌍 Remote Work Job Market Analysis

An in-depth exploration of global remote work trends, salaries, and demand across industries — powered by data analytics and visualization.


## 📌 Overview
This project analyzes remote job postings from multiple sources to uncover:
- 📈 Growth trends in remote job opportunities
- 💼 Top industries hiring remotely
- 💰 Salary patterns by region
- 🛠️ Skills most in demand
- 📊 Year-over-year changes in hiring

---

## 📊 Dataset
- **Sources:** [Kaggle Dataset]
- **Size:** 5001rows and 20 columns
- **Format:** CSV, JSON

---

## 🔍 Methodology
1. **Data Cleaning** – Removed duplicates, standardized job titles, normalized salary data.
2. **Exploratory Data Analysis (EDA)** – Identified patterns, correlations, and anomalies.
3. **Visualization** – Created interactive dashboards using Plotly and Tableau.
